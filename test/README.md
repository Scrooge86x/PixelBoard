# PixelBoard Test Client

## Getting Started

### 1. Create venv

```
python -m venv venv
```

### 2. Activate It

Mac:

```
source venv/bin/activate
```

Windows:

```
.\venv\Scripts\activate
```

### 3. Install Dependencies

```
pip install -r requirements.txt
```

### 4. Run The Server

Mac:

```
python3 test_client.py
```

Windows:

```
python test_client.py
```
